Phausto

Use the FAUST compiler and its libraries inside Pharo — live, interactive, audio synthesis.

Pharo FAUST Live Coding DSP GPL-2.0

Phausto is a library and API that brings the FAUST (Functional Audio Stream) functional audio programming language into Pharo Smalltalk. It lets you build audio synthesis chains — oscillators, filters, effects, samplers, physical models — directly from the Pharo environment, evaluated live.

Metacello new baseline: 'Phausto'; repository: 'github://lucretiomsp/phausto:main'; load.

Quick Example

"A djembe physical model through a GreyHole reverb"
pulse := Pulsen new period: (LFOTriPos new freq: 0.2; amount: 4).
djembe := Djembe new trigger: pulse.
dsp := (djembe => GreyHole new) stereo asDsp.
dsp init.
dsp start.
dsp displayUI.
dsp stop.

Architecture

Phausto compiles FAUST DSP programs to native code at runtime. Each Unit Generator (UGen) corresponds to a FAUST module. UGens are composed with Pharo's natural message-passing syntax and the => ChucK operator, then wrapped in a DSP object that manages the audio lifecycle.
